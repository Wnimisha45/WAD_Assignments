# Assignment 4a: Implementation of jQuery Mobile

## Title
Implementation of jQuery Mobile

## Problem Statement
Create a simple Mobile Website using jQuery Mobile.

## Objective
Apply jQuery Mobile library conventions and demonstrate a website.

## Files
- `index.html` - Main multi-page jQuery Mobile website
- `style.css` - Custom styling enhancements

## jQuery Mobile Conventions Used
- `data-role="page"`
- `data-role="header"`, `data-role="footer"`
- `data-role="listview"`
- `data-role="navbar"`
- `data-role="collapsible"`
- Mobile-friendly form controls

## How to Run
1. Open `index.html` in a web browser.
2. Use links/buttons to navigate between pages (Home, About, Services, Contact).

For best results, test in browser mobile view (responsive mode).
